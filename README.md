# Vercel Telegram Bot

Base on [Telegraf](https://github.com/telegraf/telegraf) framework.

## Development
### Before you start

Create `.env.development` file

```
BOT_TOKEN = "xxx"
TELEGRAM_SECRET_TOKEN = "pDgYWdoB5XqtBEC89"
GITHUB_API_TOEKN = "xxx"
CHAT_ID = "xxx"
DEBUG = "bot*,axios:*,telegraf:main,pikpak"
NODE_ENV = "development"

```

### Start your local server

```
yarn
yarn dev
```
